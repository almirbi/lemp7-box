Lemp7 Box
==========

Lemp7 Box is a preconfigured Vagrant Box with a basic setup for a web server. It runs on Ubuntu 14.04, and has only the essential tools installed:
- Ubuntu 14.04 LTS (Trusty Tahr)
- MySQL
- PHP 7.0
- Git
- cURL
- Mcrypt
- PHP GD
- WordPress

## Installation Instruction

```bash
git clone https://github.com/almirbi/lemp7-box.git project-name
cd project-name
Vagrant up
```

Visit http://192.168.33.24
